Welcome!
22 May 2013

Hey everyone! It’s time for version 2495824.225 of my blog/website. It’s pretty simple this time. Each blog post has no comments, just a title, date, and the post itself. The layout’s isn’t flashy at all because that’s what web design in 2013 means (and because I’m not very good at making pretty complex things).

Instead of using Wordpress like I’ve basically always done to create the blog, I’m using Jekyll this time. Jekyll is a “static blog generator”. In order words, so the site comes out feeling like every page was manually coded except it obviously wasn’t because the one thing developers do is make everything a shortcut. To write a blog post, I type it in a text document, build jekyll, and my site’s outputted from that1.

  1. Yes, I realize that that Jekyll isn’t that new now and I’m not exactly sure why I’m explaining how Jekyll works. ↩
