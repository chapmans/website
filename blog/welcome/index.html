Welcome!
22 May 2013

Hey everyone! Here’s my fairly simple blog/website. I wanted to make something that wasn’t flashy (because that’s basically how web design in 2013 seems to work and because I’m just bad at making complex designs) because my goal was to make something readable and navigable.

Instead of using Wordpress like I’ve basically always done to create the blog, I’m using Jekyll this time. Jekyll is a “static blog generator”. In order words, the site comes out feeling like every page was manually coded/written (except it obviously wasn’t because the one thing developers do is make everything a shortcut). To write a blog post, I type it in a text document, type “jekyll build” from the command line, and it’s then outputted into something that looks like this blog post.

With this blog, I’ll be talking about my views on this tech society that we have today – a world of tweets, iPhones, minimal design, venture capitalists, nonexistent anonymity/privacy, etc. I think that this will be more fun than having me attempt to give tutorials on Javascript or Vim or some other programming language/tool. (Besides, there’s more than enough resources online for this stuff.)
